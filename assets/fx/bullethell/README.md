# Bullet Hell generated raster assets

Generated on 2026-07-12 with the local Grok Imagine / SuperHeavy workflow.
The original 1024px outputs remain in their Grok session directories. Shipping
copies were resized with Lanczos and encoded as quality-84–86 WebP.
The four boss copies and mobile paw-bomb icon key the pure-black source into a
soft alpha edge (`yuva420p`) for `NormalBlending`/CSS; the three tintable
combat sprites intentionally keep black for additive blending.
